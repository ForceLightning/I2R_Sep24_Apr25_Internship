  • mode (Literal['train', 'val', 'test']) – Runtime mode.

  • classification_mode (ClassificationMode) – Classification mode for the dataset.

  • loading_mode (LoadingMode) – Determines the cv2.imread flags for the images.

  • combine_train_val (bool) – Whether to combine the train/val sets.

  • with_edge (bool) – Whether to return edgemaps.

  • image_size (int | Tuple[int, int]) – Output image resolution.

  • NotImplementedError – If the classification mode is not implemented.
